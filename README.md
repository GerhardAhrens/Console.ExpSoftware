# Gedanken zur heutigen Softwareentwicklung

Softwareentwicklung ist heute teuer und aufwendig. Das fängt mit der Betrachtung von Sicherheitsaspekten an, und geht weiter in die Auswahl von passenden NuGet-Paketen bis hin zur Erhaltung und Funktionfähigkeit über den Zeitraum der Verwendung.

Ein Gedanke dazu ist eine **Hybride Softwareentwicklung**, wobei es nicht um die Methodik einer Entwicklung geht, sondern um die konkrete Programmierung.
